# Changelog

## 1.0.0-rc1

- Added first-class directory fuzzing.
- Added raw request fuzzing with Burp-style request files.
- Added named placeholders and sniper/pitchfork/clusterbomb modes.
- Added ffuf-style matchers and filters.
- Added auto-calibration profiles.
- Added recursion controls.
- Added resume support.
- Added delay, jitter, max error, and max hit controls.
- Added HTTP/SOCKS proxy handling and replay proxy.
- Added JSONL and CSV exports.
- Added packaging metadata and `fuzzphantom` console command.
- Added tests and local benchmark harness.
